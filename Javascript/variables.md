
# Variables

Esta es la documentacion de variables 

## Tipo de variables 

- `const`: Variable constante, que una vez inicializada no se puede cambiar su valor
- `let`: Variable que puede cambiar su valor

```
const pi = 3.1416
let nombre = "profe"
...
nombre = "solange"
```
